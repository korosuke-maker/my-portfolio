I wondered again what kind of
place the web is.
- 「webサイトってどんな場所だろう？」改めて見つめなおしてみました。 -
It is a place that can be accessed and connected anywhere.
- 色んな人がどこにいてもアクセスでき、繋がりをつくれる場所だと思いました。 -
Someday , I was also passionate about creating such a place.
- そんな場所を私もつくりたい、気づけばweb制作に夢中になっていました。 -

Concept

webを通して企業と人、人と人との繋がりを作る

昨今では、コロナ禍でたくさんの方々の活動が留まる状況がありました。

そんな中で、web制作という分野に興味を持ち、勉強していくうちにwebを通して人と人とが繋がれる架け橋を僕はつくりたいという思いが芽生えました。

人々が直接出会う事が難しい状況があって、日常の空間が第一の空間だとしたら、webが繋がるための第二の空間であると改めて再認識しました。

全ての方々や企業には色んな可能性があって、それを上手く表現できるかできないかによって繋がる数や活動の幅は変わるかと思います。

その中で少しでも表現の選択肢になれるように、私も日々精進してweb制作で色んな方の "想い" を形にできたらと思います。

Skill

HTML-5
自己評価
HTMLに関しては、構造やその中の要素が誰が見ても分かりやすいクラス名をつける事を意識し、普段はBEM(MindBEMding)の命名規則を採用しております。また今後、HTMLのテンプレートエンジンであるpugやEJSに関して勉強して、必要であれば積極的に利用していきたいです。
CSS-3
自己評価
普段、CSS拡張言語であるSassをSCSS記法で記述しております。また、CSS設計に関しては、BEM設計（吉本式）あるいは、FLOCSS(フロックス)設計手法で最小単位のコンポーネントを意識したフォルダファイル分けをしております。デザインデータからピクセルパーフェクトで再現する練習をしてきました。
javascript
自己評価
主にjQueryライブラリを用いて記述する事がほとんどです。生のjavascriptを書けたら読み込み速度も速くてさらに汎用的だと思う事もありますが、まずはweb制作においてjQueryで制作できる、よくある挙動やコンテンツを確実に表現できる事をめざしてます。当サイトもほとんどがjQueryで書いてます。
PHP
自己評価
普段、MAMPウェブサーバーを採用して、サーバーを立ちあげております。配列や繰り返し処理などは、WordPress制作でもよく使う処理なのでしっかり記述できるように練習してます。余裕があればSQLでデータベース作成して、PHPでコンタクトフォームのデータ送受信や予約登録サイトなど練習できればWordPressのデータの動きなども理解が深まるのではないかと思います。
WordPress
自己評価
WordPressのテーマ制作を現在練習しております。1サイト静的コーディングからテンプレート化まで行い完成しました。ですが、まだまだ練習不足が感じられるので直近でWordPressサイトの練習量を増やしていきます。最近は、WordPress化を考慮したコーディングを意識するようになり、FLOCSSに基づいた使い回しが効くブロック単体でのファイル分け、ブロック内のテキスト量が増えたときやどんなサイズのimage画像がきても画角を保つように埋め込むなどデザイン崩れのないようにコーディングできる事を意識してます。
Git,制作環境
Gitは、SourceTreeツールを用いてリモート(github)に接続、ローカルにデータをプル、ブランチを作成して編集データをコミット、リモートにプッシュしプルリクエストを送る、のような基本操作ができます。GitのGUIコマンドも現在勉強しております。また、CSSコンパイル、圧縮、ブラウザ立ち上げ、更新等などはNode.jsパッケージのgulp自動化ツールを使用しており、タスクを常時監視できるよう独自で作成したgulpfile.jsを用いております。
